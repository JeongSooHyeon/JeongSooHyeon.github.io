### 2. Optional

#### 개념
- **null일 수도 있는 값을 한 번 더 감쌈**
- **NPE(NullPointerException) 방지**
- null 타입이 들어왔을 때 안전하게 대처하기 위해 만들어진 **래퍼 클래스**

#### 주요 메서드

##### isEmpty()
- 비어있으면 true

##### isPresent()
- 들어있으면 true
- **보통 이것을 사용**
- **값이 존재하는지 꼭 확인**

##### orElse()
- null일 때 기본값을 지정
- 값을 초기화할 때 바로 사용

##### orElseThrow() ⭐
- null이면 예외 강제 발생
- 파라미터: 람다식
- **보통 orElseThrow로 예외를 던짐**

##### ifPresent()
- 값이 존재할 때 파라미터로 들어오는 람다식 실행

#### 값 처리 전략

실패했을 때 3가지 방법:
1. `get()`으로 가져오기
2. `orElse()`로 초기값 넣기 (잘 안 씀)
3. **`orElseThrow()`로 예외 던지기** (권장)

### 4. 데이터 분석과 리포트 생성

### 그룹화 (Grouping)

#### groupingBy()
- 단순 그룹화
- **기준점이 되는 필드를 키로 사용**
- `collect()`로 그룹화

```java
collect(Collectors.groupingBy(...))
```

#### mapping()
- 필요한 형태로 변경 (예: 이름만 가져오기)
- 파라미터: (필요한 데이터, 형태)

#### 사용 예시
- 학년의 평균 점수 구하기 등
- 요구사항이 훨씬 복잡해지거나 데이터가 커지면 다른 언어 사용할 듯

### 7. 통계 및 집계 ⭐

> 연산하기 위해서 **특정 타입의 스트림으로 변환**해야 함

#### 기본 집계 메서드
- `sum()`, `average()`, `max()`, `min()`
- **일반적인 스트림에서 못 씀**
- `IntStream`처럼 연산이 필요한 타입이 보장되어 있는 곳에서만 가능
- 숫자를 요소로 가지는 데이터 스트림에서만 사용

#### reduce()
- 요소들을 하나의 값으로 결합
- 누적해서 연산식을 적용
- 누계를 반환
- 파라미터: 초기값과 계산식

```java
stream.reduce(0, (a, b) -> a + b)
```

#### summarizingInt()
- **count, sum, min, average, max를 한 번에 계산**
- `getSum()` 등으로 불러올 수 있음
- `IntSummaryStatistics` 형으로 반환
- 활용도가 많지는 않음
